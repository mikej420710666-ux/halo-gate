# Halo Gate Frontend

Next.js frontend for the anti-scam security toolkit.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Build

```bash
npm run build
npm start
```

## Tech Stack

- Next.js 15
- React 19
- TypeScript
- Tailwind CSS
